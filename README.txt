
Proyecto: Sitio Web de Ingeniería Multimedia

Estructura:
- index.html: Página principal con definición de Ingeniería Multimedia.
- galeria.html: Galería de imágenes con slider funcional en JavaScript.
- ventajas.html: Listado de ventajas y desventajas de estudiar esta carrera.
- unad.html: Página con información sobre estudiar Ingeniería Multimedia en la UNAD.
- /css/estilos.css: Estilos globales del sitio.
- /js/funciones.js: Script para galería y menú desplegable.
- /imagenes/: Carpeta con imágenes usadas en la galería.

Cómo usar:
1. Descomprime el ZIP.
2. Abre index.html en tu navegador.
3. Navega con el menú para visitar todas las secciones.
